大纲

  1. 本学期的要点：出师任务，要能自己独立完成项目，从研究到实践。supervised独立思考，并且用多模态来确保全面技能的养成。任务结构：
    • 2 + 2团队：2人当汇报人（老师），2人当演奏人（学生）
      • 二阶配对：组之间的自由配对
    • 任务组成：
      • 第2-9周每周一组汇报：
        • PPT汇报（30分钟）：课堂汇报，给班级讲解一首双钢琴曲目，注意曲目 + 话题的有机结合。讲解一个曲目要从切口小的话题来讨论，否则太泛（为论文写作的查重做准备）
          • PPT框架（schema，也是为之后的论文做准备）：
            • 曲子纵向所处的上下文，时代背景
            • 曲子横向所处的上下文，也就是这一曲目在双钢琴领域中的意义
            • 曲目的分析
        • 组织讨论（15分钟）：整理出讨论问题，在班级中讨论
        • 模拟教学 + 点评（双钢琴大课）（45分钟）
      • 第10周课上提交：
        • lab report: PPT汇报的论文版本
  2. 团队合作：2人小组在学术项目中的分工
    • 收集素材 + 幻灯片制作
    • 收集素材中：外部资料 + 谱面分析
  3. presentation的准备过程schema：
    • 调研
    • 整理
    • 幻灯片构思
    • 幻灯片制作
    • 内部测试：讲述并且判断主次是否合适
  4. 抽签和话题认领。平衡机制：
    • 早present的准备时间少，但是可选的话题更多
  5. presentation分为两个部分：
    • lecture：讲授。选题schema：选取（曲目，话题）这个2-tuple并且搭配
      • 上下文：曲目的时代背景，在作曲家生涯中的位置
      • 曲目作为双钢琴曲目的特色
      • 往熟悉的话题靠：如果不了解历史，那么就用曲目的具体分析来试图跟自己独奏学习学到的音乐史或者是曲目上面靠
        • 具体的分析：尤其是双向关系，乐谱上的表情记号和双钢琴的媒介如何相互影响，和促进，例如双钢琴如何助力音乐表达，乐谱上的设计如何让双钢琴的配合更加有默契
        • 比较研究，永远都有话题，因为异同都有得讲
          • 比较研究的纵横：纵就是改编曲跟原版的比较，横就是“借鉴”之间的比较
    • lesson：志愿者弹奏，然后结合讲授内容来点评和排练（训练具体应用音乐知识）
  6. presentation的选曲也会成为下半学期展演的曲目：新训练模式，全模态覆盖，熟悉从文化课角度切入学习和理解，然后再上手练琴，也就是反哺，强迫自己脑子先于手
  7. 手把手教学调研：

公告

关于发布双钢琴4课程结课及出师任务的通知及管理办法（试行）

为全面贯彻党的二十大精神，响应学院“致乐崇德”的校训，积极培养中国式现代化所需的专业人才，双钢琴4课程在编订的课程大纲的基础上，结合全面人才发展新形势、新任务、新要求，修订形成《关于发布双钢琴4课程结课及出师任务的通知及管理办法（试行）》。

现印发给你们，并就有关事项通知如下。

本学期的出师任务为本课程整个学期的主要项目，围绕着对双钢琴曲目的调研和讲解展开，分为如下几个模块：

  • 汇报：PPT主题发言 + 模拟讲课
  • 写作：汇报内容复盘及整理

汇报部分. 在第3 - 10周的8周时间里，每周将会有一组负责当周课堂的汇报任务（算入平时成绩，权重待定）。汇报任务由以下部分组成：

  1. PPT讲座/主题发言（30分钟）：你们组选取一首双钢琴或者四手联弹作品（因此舒伯特D940在选择范围内，但是双钢琴版本的钢琴协奏曲不可选），并且选取一个话题/概念当作切入点，来发表一个基于PPT的讲座/主题发言。本通知的附录有一些可供参考的曲目列表，以下是一些可以参考的话题/概念：
    • 以曲目X为锚点，试作双钢琴和其他形式室内乐的对比研究
    • 从曲目X看双钢琴艺术形式的沿袭与发展
    • 曲目X借用双钢琴形式所表达的音乐内涵与外延
    • 从曲目X试论双钢琴形式对变奏曲式的表达与实践
    • 以曲目X为案例试论Y世纪中Z主义视域下的合作艺术
  2. 组织课堂讨论（15分钟）：你们组需要为班级准备5个开放式的讨论问题作为对刚刚PPT讲座的延伸。这些问题需要能够反映刚刚PPT讲座中一些值得持续探讨的问题，尽量追求立意高，切口小。以下是一些可以参考的问题：
    • X曲目跟它的乐队原版相比，运用双钢琴配器的特点，获得了哪些新的音乐表现力？丢失了哪些？
    • 如果把这一首曲目简化成单钢琴的版本，会遇到哪些挑战？
    • 这首曲目的双钢琴配器给演奏者带来了哪些挑战或者是便利？
    • 这首曲目利用双钢琴让我们重新认识到了哪些钢琴演奏的特点？尤其是在独奏中容易疏忽的特点？
  3. 模拟讲课（30分钟）：结合PPT的内容，你们需要跟另外一个小组合作，请他们客串当“钢琴家”，视奏PPT讲座的曲目。你们可以利用他们的视奏来举例说明刚刚PPT讲座中的一些论点，因为有了视奏，所以你们可以更加深挖细节。也就是说，不要只是泛泛而谈的点评对方的演奏，而是需要将自己的点评和刚刚PPT讲座的主题结合起来

写作部分. 在第10周课上，我将会收集你们的写作项目（算入期中考试分数，权重待定）。写作项目类似于理工科的实验报告（lab report），用论文的形式来记录自己PPT讲座的内容。所以，在内容方面写作项目跟汇报部分是相同的，但是你需要用书面文字来完成，在这一过程中你需要尤其注意行文的结构性，用词避免口语化，论点论据的完整表述等等“写作课”上需要注意的要求。写作部分的项目需要以个人（而不是小组）为单位独立完成：每个人都需要提交一份自己写的完整论文。论文的硬性要求为：1200字及以上。除此之外，表述的清晰，结构的规整，行文的工整，论述的逻辑性等都是考核的重要非量化指标。

汇报顺序. 汇报的顺序由微信群抽签决定，规则如下：

  1. 需要从n组人中选取幸运观众，骰子最大点数为6，取n和6的最小公倍数φ。抛骰子$\frac{\varphi}{6}$次，并做点数求和，得到点数Σ。做取模运算(Σ mod  n) + 1，即为抽中的组的编号。班级目前的组别编号分为别：
    • 编号1：贺 龙
    • 编号2：周 龙
    • 编号3：陈 张语某
    • 编号4：药 梁
    • 编号5：陈 胡
    • 编号6：徐 葛
    • 编号7：陈 张慧某
    • 编号8：柯 翟
  2. 从初始条件即剩8组人开始，每次抽签抽到的幸运观众分别得到第3到第10周的汇报序号，并退出抽签，直到每一组都抽到了自己的序号（注意，编号跟序号不是同一个概念）。算法展开为：
    • 确定第3周汇报人选：当剩余组数 n = 8，φ = 24，需要抛骰子4次
    • 确定第4周汇报人选：当剩余组数 n = 7，φ = 42，需要抛骰子7次
    • 确定第5周汇报人选：当剩余组数 n = 6，φ = 6，需要抛骰子1次
    • 确定第6周汇报人选：当剩余组数 n = 5，φ = 30，需要抛骰子5次
    • 确定第7周汇报人选：当剩余组数 n = 4，φ = 24，需要抛骰子4次
    • 确定第8周汇报人选：当剩余组数 n = 3，φ = 6，需要抛骰子1次
    • 确定第9周汇报人选：当剩余组数 n = 2，φ = 6，需要抛骰子1次
    • 确定第10周汇报人选：当剩余组数 n = 1，达到边界条件（terminating condition/base case/boundary condition），不需要抛骰子

在线上确定汇报顺序之后，我们会在第二周的课上确定每一组的“钢琴家”搭档组。
